「プログラミングに興味はあるけど、何から始めればいいかわからない」。看護師として働きながら副業を始めようとしていた私も、まさにそうでした。

結論から言うと、**最初の1行は5分で書けます**。本記事では、Google Apps Script（GAS）の開発画面を開くところから、実際にコードを動かして結果を見るまでを順を追って解説します。

## GASとは？3行で

- Googleが提供する**無料のプログラミング実行環境**
- スプレッドシート・Gmail・カレンダーなどのGoogleサービスを自動操作できる
- 書いたコードは**Googleのサーバー上で動く**ので、パソコンに何もインストールしなくてOK

つまり、「JavaScriptの文法で、Googleサービスを操作するコード」を書くイメージです。

## 準備するもの

- Googleアカウント（1つでOK）
- ブラウザ（Chrome推奨）

**以上です**。有料ツールも、インストールも、セットアップも不要です。

## ステップ1: スクリプトエディタを開く

1. Googleドライブ（[drive.google.com](https://drive.google.com/)）にアクセス
2. 左上の「**＋ 新規**」をクリック
3. 「**その他**」→「**Google Apps Script**」を選択

新しいタブで「無題のプロジェクト」という名前のエディタが開けば成功です。

もし「Google Apps Script」が見当たらない場合は、「**アプリを追加**」から追加してください。

## ステップ2: 最初のコードを書く

エディタに以下のコードが最初から入っています：

```javascript
function myFunction() {

}
```

この `myFunction` の中身を**1行だけ追加**します。

```javascript
function myFunction() {
  Logger.log('はじめてのGAS！');
}
```

`Logger.log()` は「**実行結果を画面に表示する**」命令です。今回は文字列「はじめてのGAS！」をログに出力するだけのシンプルなコードです。

## ステップ3: 保存して実行

1. エディタ上部の「💾 保存」アイコン（または `Ctrl+S`）で保存
2. 「▶️ 実行」ボタンをクリック
3. 初回は**権限確認ダイアログ**が出るので、Googleアカウントを選んで「許可」

下部の「実行ログ」に `はじめてのGAS！` と表示されれば、**初めてのGASコード実行成功**です。

## ありがちな3つのつまずきポイント

### 1. 「実行」ボタンが押せない
保存していないと押せません。`Ctrl+S` を先に。

### 2. 権限確認で「このアプリは Google で確認されていません」と出る
個人のGASスクリプトなのでこの表示は正常です。「**詳細** → 安全でないページに移動」→「許可」で進めます。

### 3. Logger.log の結果が見当たらない
最近のGASでは「**実行ログ**」タブ、旧画面では「**表示** → ログ**」から確認できます。

## この先のステップ

「1行動いた」の次はこんな自動化が作れます：

- **Gmailの自動返信**（5分で作れます）
- **スプレッドシートの自動集計**（毎朝自動実行）
- **LINEに毎朝の天気を送信**
- **フォーム送信時にSlack通知**

どれも「Googleアカウントだけ」で作れる自動化です。
